# Analisis de algoritmos
Trabajo nÂ° 1 - Realizar mediante backtracking el recorrido de una matriz, encontrando el coste minimo entre las diferencias de cada casilla recorrida, desde 0,0 a M-1,N-1.
## Ejemplo:
1 2 3
8 7 4
9 9 5
El resultado obtenido de dicho algoritmo es de: 4, dado que el coste minimo entre las diferencias de las casillas es de 1, partiendo hacia la derecha y despues abajo.
